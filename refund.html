Refund & Cancellation Policy

We want you to be happy with Garage Book Pro Premium. Please read this policy carefully before purchasing.

1. How Premium Works Today

Garage Book Pro Premium currently costs ₹250/year and is activated using a redeem code, issued after payment is received (e.g. via UPI). The code unlocks unlimited vehicles, Full Vehicle Report, Compare, and Cloud Backup for 365 days.

2. Before You Redeem — Refunds Available

If you've paid but not yet entered your redeem code in the app, you're eligible for a full refund. Just contact us within 7 days of payment and we'll refund you, no questions asked.

3. After You Redeem — No Refund

Once a redeem code is applied to your account, Premium features are unlocked instantly and the code is permanently marked as used. Because this is a one-time digital activation, redeemed codes are non-refundable.

If a code was applied by mistake or you believe there's been an error, contact us immediately — we review these on a case-by-case basis.

4. Cancellation / Auto-Renewal

Garage Book Pro Premium does not currently auto-renew automatically via card. Your Premium access simply expires 365 days after activation. You'll need to purchase and redeem a new code to continue Premium access — there's nothing to "cancel."

5. How to Request a Refund

  • Email tcsrstorage@gmail.com with your registered email and payment reference/screenshot.
  • Refunds, once approved, are processed within 5–7 business days to your original payment method.

6. Future Payment Gateway

As we move to automated payment gateway checkout, this policy will be updated to reflect gateway-specific refund timelines. We'll notify users of any changes here.
